# IMDb Movie Review Sentiment Analysis

A Machine Learning web application built with Streamlit that predicts the sentiment of IMDb movie reviews as **Positive** or **Negative** using the **Naive Bayes algorithm**.

The project demonstrates the complete machine learning workflow, including data preprocessing, text vectorization using TF-IDF, model training, evaluation, and deployment. Users can enter a movie review through an interactive web interface and instantly receive a sentiment prediction along with the model's confidence score.

## Features

* Predicts sentiment of IMDb movie reviews
* Built using the Naive Bayes classification algorithm
* TF-IDF based text feature extraction
* Interactive Streamlit web interface
* Real-time sentiment prediction with confidence score
* Sample positive and negative reviews for testing

## Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy
* Streamlit
* TF-IDF Vectorizer
* Multinomial Naive Bayes

## Model Performance

* Training Samples: 10,000
* Test Accuracy: 81.40%

This project serves as a beginner-friendly end-to-end NLP and Machine Learning application showcasing sentiment analysis on movie reviews.
